## Hotkeys

* `Ctrl` + `A` - Select all
* `Ctrl` + `C` - Copy
* `Ctrl` + `V` - Paste
* `Ctrl` + `X` - Cut
* `Ctrl` + `O` - Open file
* `Ctrl` + `S` - Save file
* `Ctrl` + `P` - Get pixel array
* `Ctrl` + `I` - Invert layer
* `Ctrl` + `H` - Hide/Show layer
* `Ctrl` + `E` - Merge all visible layers
* `Ctrl` + `D` - Remove layer
* `Ctrl` + `J` - Clone layer
* `Ctrl` + `+` - Zoom in
* `Ctrl` + `-` - Zoom out
* `Ctrl` + `0` - Initialize zoom
* `Shift` + `N` - Create new layer
